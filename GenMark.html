GENMARK

> Procedural Generation Algorithm evaluation tool

// OVERVIEW.TXT

GenMark is an evaluation tool for procedural content generation (PCG) in video games. It provides a framework for measuring PCG algorithms both computationally and in terms of player engagement, helping developers assess and improve the quality of generated content.

// SYSTEMS.LOG
// MAP_GEN.SYS

2D Perlin noise-based, tile-driven map generator used as a testbed for evaluating GenMark’s PCG metrics.

// PARAM.SYS

Parameter-driven generation system controlling map output across three metrics: quality (structural coherence), diversity (tile variety and distribution), and controllability (the degree to which user-defined constraints and tile ratios influence the final output).

// EVAL.SYS

Evaluation system that uses A* pathfinding between two random points to analyze map solvability, path efficiency, and tile diversity along traversal paths to approximate player engagement.
