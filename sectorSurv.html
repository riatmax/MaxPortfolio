> 2D cutsie violence shooter action

// OVERVIEW.TXT

This project is a 2D game developed in Java using a custom game engine provided as part of a coursework framework, with the primary goal of extending a minimal base engine into a fully functional gameplay prototype through the implementation of core systems and interaction mechanics. My contributions included designing and implementing a wave-based enemy spawning system to control structured progression and difficulty scaling, adding full mouse input support to enable precise player interaction with gameplay elements, and developing an inventory system for managing item collection, storage, and usage. Together, these systems form a cohesive gameplay loop that demonstrates experience working within a constrained engine architecture, extending existing frameworks, and integrating multiple gameplay systems into a unified interactive experience.

// SYSTEMS.LOG
// WAVE.SYS

Wave-based spawning system responsible for managing enemy progression. Controls spawn timing, wave scaling, and difficulty escalation to create structured gameplay pacing and increasing challenge over time.

// INPUT_MOUSE.SYS

Custom mouse input handling layer built on top of the base engine. Enables real-time interaction with game objects, supporting click detection and in-game shooting mechanics.

// INVENTORY.SYS

Inventory management system handling item storage and retrieval. Provides a structured framework for player resource management and supports integration with gameplay mechanics such as item-based interactions and progression.
